│ STEP 1: Add UTF-8 Support (Top of file)                                  │
├───────────────────────────────────────────────────────────────────────────┤
│ <%@ page contentType="text/html; charset=UTF-8" pageEncoding="UTF-8"%>   │
│ <%@ page import="com.election.i18n.MessageBundle" %>                     │
└───────────────────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────────────────┐
│ STEP 2: Add Language Selector (In navbar)                                │
├───────────────────────────────────────────────────────────────────────────┤
│ <jsp:include page="/includes/language-selector.jsp" />                   │
└───────────────────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────────────────┐
│ STEP 3: Replace Hardcoded Text                                           │
├───────────────────────────────────────────────────────────────────────────┤
│ BEFORE: <h1>Dashboard</h1>                                               │
│ AFTER:  <h1><%= MessageBundle.getMessage(request, "app.dashboard") %></h1>│
│                                                                            │
│ BEFORE: <button>Save</button>                                            │
│ AFTER:  <button><%= MessageBundle.getMessage(request, "action.save") %>  │
│         </button>                                                         │
└───────────────────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────────────────┐
│ HOW IT WORKS:                                                             │
├───────────────────────────────────────────────────────────────────────────┤
│ 1. User types Hindi/Marathi in input field: "राजेश कुमार"               │
│ 2. CharacterEncodingFilter converts to UTF-8                             │
│ 3. Servlet receives correctly encoded string                             │
│ 4. Saves to database (utf8mb4 charset)                                   │
│ 5. Retrieves from database correctly                                     │
│ 6. Displays on page: "राजेश कुमार" ✓                                    │
├───────────────────────────────────────────────────────────────────────────┤
│ REQUIREMENTS (ALL ALREADY IMPLEMENTED):                                  │

│ ✅ JSP UTF-8 declaration                                                 │
│ ✅ Proper fonts (Noto Sans Devanagari)                                   │
