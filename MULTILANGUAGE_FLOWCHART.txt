┌──────────────────────────────────────────────────────────────────────────────┐
│                         8. DATA FLOW SUMMARY                                  │
└──────────────────────────────────────────────────────────────────────────────┘

    User Login → Select Language → Store in Session
                          │
                          ▼
              Dashboard Loads → Reads Session Language
                          │
                          ▼
              JSP Calls MessageBundle.getMessage()
                          │
                          ▼
              MessageBundle loads appropriate .properties
                          │
                          ▼
              Returns translated text to JSP
                          │
                          ▼
              UI renders in selected language
                          │
                          ▼
              User can switch language anytime via navbar
